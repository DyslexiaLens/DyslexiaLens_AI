# DyslexiaLens_AI
Infrastruktur backend produksi DyslexiaLens berbasis FastAPI dan Docker, mengintegrasikan arsitektur Multi-Modal Late Fusion CNN untuk deteksi disleksia dan skor keparahan, pemrosesan citra grid kustom (Traditional CV &amp; OCR), serta generator kalimat otomatis bertenaga Generative AI.
